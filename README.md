Dot Files
=====

Installation
============
    $ git clone https://github.com/khaledez/dotfiles.git ~/.dotfiles
    $ chmod +x ~/.dotfiles/install && ~/.dotfiles/install

VIM
============
I'm using Vundle for plugin management.

Inside Vim:

    :PluginInstall

To update plugins:

    :PluginUpdate

Clean plugins:

    :PluginClean
