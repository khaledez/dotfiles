vimrc
=====

My .vimrc configuration

Installation
============
    $ git clone --recursive https://github.com/khaledez/vimrc.git ~/.vim
    $ ln -s ~/.vim/vimrc ~/.vimrc
    $ vim +PluginInstall +qall

Inside Vim:

    :PluginInstall

To update plugins:

    :PluginUpdate

Clean plugins:

    :PluginClean
